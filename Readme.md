# 📐 Calculadora de Exercícios Matemáticos

Aplicação desenvolvida com **HTML**, **CSS** e **JavaScript** para resolver uma série de exercícios matemáticos e de lógica propostos em atividades acadêmicas de programação.

## 🚀 Funcionalidades

O sistema realiza os seguintes cálculos:

### 🌡️ Conversão de Temperatura

* Conversão de **Celsius para Fahrenheit**
* Conversão de **Fahrenheit para Celsius**

### 🛢️ Volume de Objetos

* Cálculo do volume de uma **lata de óleo**
* Cálculo do volume de uma **caixa retangular**

### ⛽ Consumo de Combustível

Calcula:

* Tempo gasto na viagem
* Velocidade média
* Distância percorrida
* Litros de combustível consumidos

Considerando um veículo com consumo médio de **12 km/L**.

### 💰 Prestação em Atraso

Calcula o valor atualizado de uma prestação utilizando:

* Valor original
* Taxa de juros
* Tempo de atraso

### 🔄 Troca de Valores

Realiza a troca de valores entre duas variáveis:

* A recebe o valor de B
* B recebe o valor de A

### ➕ Operações Combinatórias

A partir de quatro valores inteiros (A, B, C e D), o sistema realiza:

* 6 adições
* 6 multiplicações

Utilizando todas as combinações possíveis entre os valores.

### 🔢 Potenciação

* Cálculo do quadrado de um número inteiro.
* Cálculo do quadrado da diferença entre dois números inteiros.

## 📦 Como Executar o Projeto

### 1. Clone o repositório

```bash
git clone https://github.com/Henzo-Brito/mini-projects.git
```

### 2. Acesse a pasta do projeto

```bash
cd mini-projects
```

### 3. Inicie um servidor local

Como o projeto utiliza módulos JavaScript, é necessário executá-lo através de um servidor HTTP:

```bash
npx serve .
```

### 4. Abra no navegador

Após iniciar o servidor, acesse o endereço exibido no terminal, normalmente:

```txt
http://localhost:3000
```
